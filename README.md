💰 SpendWise — Smart Expense Tracker
<br>
🚀 Live Demo: https://spend-wise-lemon-two.vercel.app/
<br>
📌 Overview

SpendWise is a modern expense tracking web application designed to help users take control of their finances through clear insights, structured tracking, and intuitive design.

The app goes beyond basic expense logging by focusing on usability, performance, and real-world financial awareness.
<hr>
🎯 Problem
<p>
Managing personal finances can be overwhelming without visibility into spending habits.
Many existing tools are either too complex or lack meaningful insights.
</p>
<hr>
💡 Solution

SpendWise provides a simple yet powerful interface to:

Track daily expenses
Organize spending into categories
Monitor financial habits over time
✨ Key Features
➕ Add, edit, and delete expenses
🗂 Categorize transactions (e.g., Food, Transport, Bills)
📅 Track expenses by date
📊 Visual insights for better decision-making (charts/analytics — if added)
🔍 Filter and view spending patterns (if implemented)
🔐 User authentication 
☁️ Persistent data storage
🛠 Tech Stack

Frontend:

React
JavaScript (ES6+)
HTML5 & CSS3

Backend: Node.js

Database : MYSQL

Deployment: Vercel

2. Install dependencies

3. Run the app
npm start
